A Machine Learning powered web application that recommends the optimal crop to cultivate based on soil conditions (Nitrogen, Phosphorus, Potassium) and climate parameters (Temperature, Humidity, Rainfall).

## 🌟 Features
- **Interactive Web Interface**: Built with Streamlit for a smooth user experience.
- **Machine Learning Model**: Utilizes a trained model to make accurate crop predictions.
- **Dynamic Inputs**: Users can easily adjust parameters using sliders and input fields.
- **Detailed Output**: Provides clear, actionable recommendations for farmers and agricultural planners.

## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/dharmander-081/crop-classification.git
   cd crop-classification
   ```

2. **Install the required dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit application**
   ```bash
   streamlit run app.py
   ```

4. **Open your browser**
   The application will automatically open at `http://localhost:8501`.

## 📁 Repository Structure
- `app.py`: Main Streamlit application code.
- `Crop_Recommendation_System.ipynb`: Jupyter notebook containing data analysis and model building.
- `crop_model.pkl`: Serialized trained machine learning model.
- `requirements.txt`: List of Python dependencies required to run the project.
- `Crop_recommendation.csv`: The dataset used for training the model.
